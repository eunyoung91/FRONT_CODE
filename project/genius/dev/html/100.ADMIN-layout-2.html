조직도 히스토리 관리를 위해 아래와 같은 기준으로 부서 생성/수정 해주세요.

1. 조직도 변경 방법 (조직 추가/삭제, 조직 명칭/정렬 변경)
 1) '변경 시작' 버튼을 클릭.
 2) 조직 추가/삭제, 조직 명칭/정렬 변경 후 '저장' 버튼 클릭.
 3) 저장 완료 후 '변경 완료' 버튼 클릭.

2. 조직 생성 / 조직 명칭 변경 기준
1) 조직 생성 기준
  - 조직이 신설된 경우
  - 조직이 분할되는 경우 (ex. IT서비스본부 서비스기획팀 삭제 → 서비스기획1/2팀 생성)
  - 조직이 분할되며, 명칭이 변경된 경우 (ex. IT플랫폼본부 인프라개발팀 삭제 → 플랫폼개발1/2팀 생성)
  - 조직이 통합되는 경우 (ex. CM1본부, CM2본부 삭제 → CM본부 생성 )
  - 단, 상위 부서가 신설되는 경우 상위 부서만 생성 후 하위부서는 정렬 변경(하위 부서 삭제 후 재생성 하지 않습니다.)

2) 조직 수정 기준
  - 명칭이 변경되지만, 조직 구성원 변경 없을 경우(ex. IT서비스본부 시스템실 네트워크보안팀 → 정보보호팀 명칭 변경)
