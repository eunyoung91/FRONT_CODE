참여방법 이벤트 기간 내 신청한 신규회원에게 선착순으로 제공됩니다. 배송비 2,500원 결제 완료 시 정상적으로 신청이 완료됩니다. 장바구니에 담겨진 교재 배송비는 7일 이내에 결제하셔야 하며, 기간 경과 혹은 장바구니에서 삭제 시 재신청이 불가합니다. 발송이 진행된 후에는 배송비 환불 불가합니다. 신규회원에 한해 신청 가능하며, 미리 회원가입 해 두시면 빨리 신청하실 수 있습니다.
