환급결과
수강료 환급 신청 시 유의사항
최종 합격 발표 후, 7일 이내에 에듀윌 사이트에서 반드시 합격수기 작성 후, 수강료 환급 신청을 하셔야 현금 환급됩니다.
수강료 환급 신청에 필요한 서류 : 응시확인서(수험표), 합격증, (주민등록번호 전체 기입)신분증, (본인명의)통장 이미지를 제출해야 합니다.
공인/주택처럼 1∙2차 분리시험인 경우, 응시확인서 이미지 : 1차합격증 / 합격증 이미지 : 2차합격증을 업로드해주세요.
기사시험처럼 필기, 시험 분리시험인 경우, 응시확인서 이미지 : 필기합격증 / 합격증 이미지 : 실기합격증을 업로드해주세요.
공무원시험 합격자는 합격수기를 에듀윌 합격수기 게시판과 아래 3개의 카페 중 한 곳의 합격수기 게시판에 등록해주시고, URL을 수강료환급신청 단계에서 기입해주세요.
  • 에듀윌 합격수기 게시판 : http://gov.eduwill.net/Gov2/promotion/victoryStory/index.asp
  • 닥공사 : https://cafe.naver.com/kts9719
  • 공드림 : https://cafe.naver.com/gugrade
  • 구꿈사 : http://cafe.daum.net/9glade
