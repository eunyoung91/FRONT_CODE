유의사항
EVENT 01
- 대상자 : 합격자 인증을 마친 모든 회원 (기존 합격자 인증자 포함)
- 적용시기 : 합격자 인증 승인 이후, 답변 개수가 카운팅 됩니다.
- 혜택 제공 : 매월 초, 답변을 확인하여 20개 이상인 합격자들에게 기프티콘이 제공됩니다. 무성의한 답변은 카운팅에서 제외되오니 참고바랍니다.
EVENT 02
- 프로필 관리에서 합격자 인증에 성공 시, 관리자가 개별적으로 내공 10,000점을 등록해드립니다.
EVENT 03
- 대상자 : 합격자를 추천한 모든 회원(합격자, 수험생 모두 해당)
- 합격자 : 에듀윌 외 타사 합격자도 무방합니다.
- 혜택 방식 : 추천한 합격자가 질문에 10개 이상 답변하면, 추천자 에게 스타벅스 커피를 증정합니다. (추천자 ID 당 1회에 한함)
- 혜택 제공 : 매월 초, 확인 후 일괄 지급됩니다.
