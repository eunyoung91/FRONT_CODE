메모 폰트/컬러/크기를 적용하는 샘플입니다.
Label
left : 차트의 왼쪽 가장자리에서 얼마만큼 Label을 위치 시킬지 나타냅니다.
right : 차트의 오른쪽 가장자리에서 얼마만큼 Label을 위치 시킬지 나타냅니다.
top : 차트 상단의 가장자리에서 얼마만큼 Label을 위치 시킬지 나타냅니다.
bottom : 차트 하단의 가장자리에서 얼마만큼 Label을 위치 시킬지 나타냅니다.
color : 글자 색상을 나타냅니다.
cornerRadius : Label에 나오는 박스 테두리 모서리의 둥근 정도를 나타냅니다.
fontFamily : 폰트 명 입니다.
fontSize : 폰트 사이즈 입니다.
fontWeight : bold로 글자를 표현할지 나타냅니다. normal | bold
backgroundAlpha : 배경색의 투명도 입니다.
backgroundColor : 배경의 색상 입니다.
borderAlpha : 테두리의 투명도 입니다.
borderColor : 테두리의 색상 입니다.
borderThickness : 테두리의 두께 입니다.
textAlign : 텍스트의 정렬을 나타냅니다. left | center | right
styleName : style노드에서 정의한 스타일을 설정 합니다.
개행처리는 &#xa; 를 사용하여주십시오.

●Layout
상단
<Label right="0" textAlign="center" text="글씨 크기 예제 입니다." fontFamily="맑은 고딕" 
textDecoration="underline" backgroundColor="0x9d4c64" fontSize="17" backgroundAlpha="0"/>
중단
<Label width="150" top="150" right="20" borderColor="0x0000ff"  text="글씨 색상 예제 입니다." 
cornerRadius="10" fontSize="11" color="0xff0000" textAlign="center" backgroundColor="0xf5d6fc"/>
하단
<Label width="200" height="60" top="230" right="20" textAlign="center" borderStyle="none"
text="색상과 크기&#xa;예제 입니다." fontSize="20" color="0xffffff" fontWeight="bold" 
borderColor="0xcd5c5c" borderThickness="2"  ackgroundColor="0xf08080" backgroundAlpha="1">
</Label>
