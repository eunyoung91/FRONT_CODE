등급정책 내공 쌓는 법
  • 1. 사용한 내공만큼 내공차감, 등급유지
  • 2. 회원정보에 등록된 휴대전화번호와 주소로 지급
    회원정보 수정
  • 3. 관리자가 부정한 방법으로 내공을 획득했다고 판단한 경우 상품은 지급불가
    (※부정참여자 처리방안은 하단 유의사항을 확인해 주세요)
부정참여자 처리방안
무의미한 게시글 등 부정한 방법으로 내공 획득 후 내공자판기를 이용한 경우 부정참여자로 분류합니다.
부정참여자는 상품 미지급 및 지급받은 내공이 차감처리 됩니다.
3회 이상 부정 참여한 경우 영구적으로 내공자판기 이용이 불가합니다.
게시글 신고하기
더 나은 에듀윌 지식인이 될 수 있도록 부정한 방법으로 작성된 게시글을 직접 신고해주세요 ^^
신고한 게시글이 부정 게시글로 판정되면, 최초 신고자에게 내공 +5점을 드립니다.
유의사항
  • 1. 내공자판기 운영 방침
    1) 내공자판기는 평민 등급 이상부터 이용 가능합니다.
    2) 에듀윌 지식인 공무원에서 획득한 내공을 상품으로 교환 할 수 있습니다.(에듀윌 지식인 공인중개사와 별도로 운영됩니다. )
    3) 각 상품마다 신청가능한 등급과 수량이 정해져 있으므로, 교환 전 본인의 내공과 등급을 확인해주세요.
    4) 내공자판기의 상품과 상품에 해당 하는 내공은 변경 될 수 있습니다.
    5) 교환에 사용한 내공은 차감되나, 등급은 유지됩니다.
    6) 부정한 방법으로 내공자핀기를 이용한 경우 최대 3회 적발 시, 영구적으로 참여가 불가능 합니다.
    7) 내공자판기 이용 후 회원 탈퇴한 경우 상품 지급이 불가합니다.
    8) 부정 게시글을 신고한 회원에게는 내공 5점이 지급되며, 해당 게시글을 최초 신고한 회원에 한해서만 제공됩니다.
  • 2. 부정참여자 관리 방침
    1) 아래의 부정한 방법으로 내공 획득 후 내공자판기를 이용한 경우 부정참여자로 분류합니다.
    - 무의미한 게시글을 무작위 작성하여 내공을 획득한 경우
    - 여러 아이디로 답변등록, 채택한 정황이 목격된 경우
    - 내공자판기 이용 후 회원 탈퇴해 재가입하여 내공 획득한 경우
    - 자판기 이용 후 질문, 답변, 댓글 게시글 삭제한 경우
    - 이 외에, 관리자가 판단 시 부정한 방법으로 내공을 획득했다고 판단되는 경우
    2) 3회 이상 부정 참여한 경우 영구적으로 내공자판기 이용이 불가합니다.
    3) 부정참여자가 내공자판기 이용 시 아래와 같이 조취 됩니다.
    - 내공자판기에서 상품 교환 신청 후 부정참여자로 판단 된 경우 상품 미지급, 부정하게 지급받은 내공 차감 처리
    - 이미 상품 지급이 완료 된 후 부정참여자로 판단 된 경우 부정하게 지급 받은 내공 차감 처리
    - 상품 지급 전 탈퇴한 회원의 경우 상품 미지급
    4) 부정참여가 적발된 경우, 에듀윌 지식인 알림을 통해 처리사항이 공지됩니다.
  • 3. 상품 지급 유의사항
    1) 교환한 상품은 교환 신청한 날의 다음주 월요일에 일괄 지급됩니다. (월요일이 휴일인 경우 휴일 다음날 지급)
    2) 상품은 회원정보에 등록된 휴대번호와 주소로 지급되며, 정보 오류로 인한 재지급은 불가하니 회원정보를 꼭 확인해주세요.
    회원정보 수정
    3) 상품은 사전 고지없이 동등한 가치를 가진 상품으로 대체되어 제공될 수 있습니다.
    4) 관리자가 부정한 방법으로 내공을 획득했다고 판단한 경우 상품은 제공되지 않습니다.
  • 4. 제공 상품
    - 백화점 상품권 : 3만원 상당의 백화점 상품권
    - 전복죽
    - 치킨 : 교촌허니오리지널 치킨
    - 버거 : 버커킹 와퍼세트
